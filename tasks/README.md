# Task Lists

Phase-by-phase task tracker for the Smart Geyser project. Source of truth for what is done, in progress, and pending.

## Conventions

- Each phase has its own file: [phase-1.md](phase-1.md), [phase-2.md](phase-2.md), etc.
- Checkbox statuses:
  - `[ ]` — not started
  - `[~]` — in progress
  - `[x]` — done
  - `[-]` — skipped / deferred (note why)
- When a task is finished, **mark it immediately** in the relevant phase file. Do not batch.
- Keep notes/decisions inline under the task they relate to.
